Promedio Móvil Simple

El promedio móvil simple es una técnica utilizada en el análisis de series de tiempo para suavizar las fluctuaciones y destacar las tendencias a largo plazo. Se calcula tomando la media de un conjunto de valores consecutivos en la serie de tiempo.

Ejemplo

Supongamos que tenemos los siguientes datos mensuales de ventas de un producto:

  • Enero: 100 unidades
  • Febrero: 110 unidades
  • Marzo: 120 unidades
  • Abril: 115 unidades
  • Mayo: 130 unidades

Si aplicamos un promedio móvil simple de 3 períodos, el resultado sería:

  • Febrero: (100 + 110 + 120) / 3 = 110 unidades
  • Marzo: (110 + 120 + 115) / 3 = 115 unidades
  • Abril: (120 + 115 + 130) / 3 = 121.67 unidades
